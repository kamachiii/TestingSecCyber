# VulnLab - Keamanan Web
## Fitur
- Autentikasi pengguna (Login/Logout)
- Dashboard pengguna
- Sistem komentar di halaman profil
- Fitur transfer saldo antar pengguna
- Halaman admin dasar

## Prasyarat
- Python 3.x
- pip

## Cara Menjalankan
1. Install dependencies dari requirements.txt

```cmd
pip install -r requirements.txt
```

2. Jalankan aplikasi

```cmd
flask run
```

3. Buka `http://127.0.0.1:5000` di browser anda
